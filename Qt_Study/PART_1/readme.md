# Qt_Study
* 基本的类
QTabWidget 菜单选项卡  
QLable 标签  
QTextEdit 多行编辑器  
QLineEdit 单行编辑器  
QPushButton 按钮  
QGroupBox 组栏  
QTableWidget 表格  
QCheckBox  选择框  
QRadioButton 单选框  
QHBoxLayout 水平布局  
QVBoxLayout 垂直布局  
* 常用的快捷键
Ctrl+H 水平布局,也可以给整个Widget水平布局，这样在窗口大小变化的时候，部件跟着变化  
Ctrl+L 垂直布局  
Shift+Alt+R 预览  
this->setMaximumSize(393,187);固定最大的窗口大小  
this->setMinimumSize(393,187);固定最小的窗口大小
